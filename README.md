## Описание программы

Expense Tracker — это графическое приложение для отслеживания личных расходов. Позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и периоду, автоматически подсчитывать общую сумму.

---

## Требования для запуска

- Python 3.10 или выше
- Дополнительные библиотеки: не требуются (используется только встроенная `tkinter`)

---

## Как запустить

```bash
git clone https://github.com/ваш-логин/expense-tracker.git
cd expense-tracker
python app.py
```

---

## Как пользоваться

1. Заполните поля: сумма, категория, дата
2. Нажмите "Добавить" — запись появится в таблице
3. Для фильтрации: выберите категорию и/или период дат, нажмите "Применить"
4. Для удаления записи: выберите строку в таблице и нажмите "Удалить выбранное"
5. Данные автоматически сохраняются в `data/expenses.json`

---

## Пример работы

Добавление расхода:
- Сумма: 500
- Категория: Еда
- Дата: 2026-05-05
- Результат: Запись добавлена в таблицу

Фильтрация по категории "Транспорт":
- Показываются только расходы на транспорт
- Итого пересчитывается автоматически

## Тестирование

| Тип теста | Пример | Результат |
|-----------|--------|-----------|
| Позитивный | Добавить расход 500 руб | ✅ Запись добавлена |
| Негативный | Ввод "abc" вместо суммы | ❌ Ошибка: введите корректную сумму |
| Граничный | Сумма = 0 | ❌ Ошибка: сумма должна быть > 0 |
| Граничный | Дата в неверном формате | ❌ Ошибка: неверный формат даты |
